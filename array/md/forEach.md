> # Array.forEach()

The forEach() method calls a function for each element in an array.

**Example:-**

```javascript
const array = [2, 4, 6, 8, 10];

array.forEach((element) => console.log(element)); //output:- 2,
// 4,
// 6,
// 8,
// 10
```
